# Kepler Exoplanet Analysis

## Overview

This project explores NASA's Kepler Exoplanet dataset using Python and data analysis techniques.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings

- The dataset contains 9,564 exoplanet candidates.
- Gas Giants are the most common planet category.
- Planet radius distribution is strongly right-skewed.
- Equilibrium temperatures span a very wide range.
- Only a limited number of confirmed planets meet the Earth-like criteria used in this analysis.
